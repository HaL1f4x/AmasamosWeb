# Instrucciones para Codex (AmasamosWeb)

Usa los archivos en /content como fuente de contenido (copy) y /assets como imágenes.

Objetivo:
- Generar un sitio estático (HTML/CSS/JS) compatible con GitHub Pages.
- Página principal: index.html
- Secciones: Inicio, Pizzamos, Amasar Institucional, Contacto.
- Dentro de Amasar Institucional dos secciones, Linea Premium y Linea Standard
- CTA principal: WhatsApp y correo.
- Estilo: limpio, moderno, artesanal/premium.

Requerimientos:
- Sitio responsive (mobile-first).
- Botón fijo de WhatsApp.
- SEO básico: title, meta description, H1 único, alt en imágenes.
- No usar backend. Solo estático.
- AI Ready

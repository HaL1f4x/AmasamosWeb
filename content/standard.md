# Línea Standard

## Descripción
La línea Standard de Amasamos está diseñada para restaurantes que buscan
calidad constante y excelente relación costo-beneficio.

## Características
- Pan brioche suave
- Buena estructura para hamburguesas
- Ideal para alto volumen

## Aplicaciones
Perfecto para restaurantes de hamburguesas y negocios de comida rápida.
